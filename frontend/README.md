# UniLib Frontend

This frontend uses React, Vite, Tailwind CSS, and `.jsx` component files.

## Scripts

`npm run dev`

Starts the Vite development server.

`npm run build`

Builds the app into `dist/`.

`npm run preview`

Previews the production build locally.

## Environment Variables

Use Vite-prefixed variables in `.env`:

`VITE_USER_SERVICE_URL`

`VITE_BOOK_SERVICE_URL`

`VITE_LOAN_SERVICE_URL`

`VITE_NOTIFICATION_SERVICE_URL`
